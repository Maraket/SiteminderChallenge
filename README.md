## Description

Tech challenge for SiteMinder

## Require

It is required that npm is installed and that yarn is available globally

## Installation

```bash
$ yarn
```

## Running the app

```bash
$ yarn run
```

## Test

```bash
# unit tests
$ yarn test
```
